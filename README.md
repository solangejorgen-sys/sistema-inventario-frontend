# Inventario Frontend

Aplicacao Vue para o sistema de inventario.

## Railway

1. Crie um repositorio apenas com os ficheiros desta pasta `frontend`.
2. No Railway, crie um novo servico a partir desse repositorio.
3. Configure a variavel `VITE_API_URL` com a URL publica do backend, por exemplo:

```bash
VITE_API_URL=https://SEU-BACKEND.up.railway.app
```

Depois de alterar `VITE_API_URL`, faca redeploy do frontend para o Vite incluir a nova URL no build.

## Local

```bash
npm install
npm run dev
```
